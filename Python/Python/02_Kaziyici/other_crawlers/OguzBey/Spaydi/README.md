# Spaydi !

Spaydi bir Web-Spider/Crawler aracıdır.

## Özellikler

- Html Formları Toplama
- Derinlik seviyesine göre Toplama 1...5
- Cookie desteği
- Link ve Form çıktısı ./outputs
- Hızlı tarama desteği
